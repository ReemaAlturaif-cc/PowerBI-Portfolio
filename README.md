# PowerBI-Portfolio
Welcome to my Power BI portfolio.

This repository contains end-to-end Power BI projects demonstrating:

- Data Cleaning (Power Query)
- Data Modeling
- Star Schema
- DAX Measures
- Time Intelligence
- Interactive Dashboards
- Business Insights

## Projects

### Sales Dashboard

Retail sales analysis using the Sample Superstore dataset.

### HR Dashboard

Employee attrition and workforce analysis using the IBM HR dataset.

### Finance Dashboard

Financial analysis using Microsoft's Financial Sample dataset.

### Brazilian E-commerce Dashboard

End-to-end e-commerce analytics using the Olist dataset.
