# Sales Dashboard | Power BI

## Project Overview

This project analyzes retail sales performance using Power BI. The dashboard provides insights into revenue, profit, customer orders, and product performance to support business decision-making.

---

## Dataset

- Sample Superstore Dataset
From kaggle.com
---

## Tools Used

- Power BI Desktop
- Power Query
- DAX
- Data Modeling

---

## Data Preparation

- Removed negative profit values.
- Created Category Hierarchy.
- Created Customer Hierarchy.
- Built a Date Table.
- Created relationships between tables.

---

## Data Model

- Star Schema
- Date Table
- Fact & Dimension tables

---

## DAX Measures

- Total Sales
- Total Profit
- Profit Margin %
- Total Orders

---

## Dashboard Features

- Executive KPIs
- Monthly Sales Trend
- Sales by Category
- Sales by Region
- Top Products
- Interactive Filters

---

## Business Insights

- Technology generated the highest sales.
- West region achieved the highest revenue.
