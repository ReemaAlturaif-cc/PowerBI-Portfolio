# Brazilian E-commerce Dashboard | Power BI

## Project Overview

This project analyzes a real-world Brazilian e-commerce dataset (Olist) using a Star Schema data model. The dashboard provides insights into sales performance, customer behavior, delivery efficiency, and payment methods.

---
## Dataset

Brazilian E-commerce Public Dataset by Olist (Kaggle)
From kaggle.com

---
## Tools Used

- Power BI Desktop
- Power Query
- DAX
- Data Modeling
- Star Schema

---
## Data Preparation

- Removed unnecessary columns.
- Removed Geolocation table.
- Renamed columns.
- Created Date Table.
- Created Order Date column.
- Built relationships.

---
## Key Business Insights

- Health & Beauty generated the highest revenue.
- São Paulo (SP) recorded the highest sales.
- Most customers used Credit Cards.
- More than 90% of orders were delivered on time.
- Most customers gave a 5-star review.
