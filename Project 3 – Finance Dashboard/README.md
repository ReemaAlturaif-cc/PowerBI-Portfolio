# Finance Dashboard | Power BI

## Project Overview

This dashboard analyzes financial performance by tracking sales, profit, costs, and year-over-year growth using advanced DAX calculations.

---
## Dataset

Microsoft Financial Sample Dataset
From Microsoft Learn

---
## Tools Used

- Power BI
- Power Query
- DAX

---
## Data Preparation

- Built Date Table.
- Created relationships.
- Organized measures into folders.

---
## DAX Measures

- Total Sales
- Gross Sales
- Profit
- Profit Margin %
- Sales YTD
- Previous Year Sales
- Previous Year Profit
- YoY Growth %
- Running Total

---
## Dashboard Features

- Sales Analysis
- Country Performance
- Segment Analysis
- Monthly Sales Trend
- Top Products
- Time Intelligence

---
## Business Insights

- Government segment generated the highest sales.
- Profit increased compared to the previous year.
- Sales growth varied across countries.
